## Overview

Website personal ini dibangun menggunakan teknologi modern untuk memberikan pengalaman yang optimal bagi pengunjung. Beberapa fitur utama website ini meliputi:

- **Blog Posts**: Tempat berbagi pengetahuan dan pengalaman dalam dunia pengembangan perangkat lunak
- **Projects Showcase**: Menampilkan berbagai proyek yang telah saya kerjakan
- **Dark Mode**: Mendukung tema gelap untuk kenyamanan membaca
- **Responsive Design**: Tampilan yang optimal di berbagai ukuran layar
- **MDX Support**: Penulisan konten menggunakan MDX untuk fleksibilitas lebih
- **SEO Optimized**: Optimasi untuk mesin pencari

## Screenshots & Features

### Halaman Utama
Halaman utama menampilkan gambaran singkat tentang saya sebagai Software Engineer. Di sini pengunjung dapat melihat proyek-proyek terbaru yang saya kerjakan dan artikel blog terbaru yang saya tulis. Desain minimalis dengan animasi halus memberikan kesan profesional dan modern.

### Dark Mode
Website ini mendukung dark mode untuk kenyamanan membaca di kondisi cahaya rendah. Perpindahan antara light dan dark mode dirancang dengan transisi yang halus untuk pengalaman yang lebih baik.

### Blog Posts
Halaman blog menampilkan artikel-artikel teknis dengan dukungan syntax highlighting untuk kode, dan MDX untuk konten yang lebih interaktif. Setiap artikel dilengkapi dengan estimated reading time dan kategori untuk memudahkan navigasi.

## Tech Stack

Website ini dibangun menggunakan stack teknologi modern:

### Frontend
- **Next.js**: Framework React untuk server-side rendering dan static site generation
- **TypeScript**: Untuk type safety dan developer experience yang lebih baik
- **Tailwind CSS**: Untuk styling yang fleksibel dan maintainable
- **Framer Motion**: Untuk animasi yang smooth dan interaktif

### Content Management
- **MDX**: Untuk menulis konten dengan komponen React
- **Contentlayer**: Untuk transformasi content menjadi type-safe data
- **rehype/remark**: Untuk pemrosesan markdown

### Development Tools
- **ESLint**: Untuk code linting
- **Prettier**: Untuk code formatting
- **Husky**: Untuk git hooks
- **Commitlint**: Untuk conventional commits

## Pembelajaran & Tantangan

Dalam proses pengembangan website ini, saya menghadapi beberapa tantangan dan pembelajaran:

1. **Optimasi Performa**: Implementasi dynamic imports dan image optimization untuk meningkatkan performance score
2. **SEO**: Penerapan meta tags dan structured data untuk optimasi mesin pencari
3. **Aksesibilitas**: Memastikan website dapat diakses dengan baik oleh semua pengguna
4. **Content Management**: Menggunakan Contentlayer untuk mengelola konten MDX dengan type safety
