## Overview

This personal website is built using modern technologies to provide an optimal experience for visitors. Some of the main features include:

- **Blog Posts**: A place to share knowledge and experiences in software development
- **Projects Showcase**: Displaying various projects I have worked on
- **Dark Mode**: Support for dark theme for comfortable reading
- **Responsive Design**: Optimal display across various screen sizes
- **MDX Support**: Content writing using MDX for greater flexibility
- **SEO Optimized**: Search engine optimization

## Screenshots & Features

### Home Page
The home page displays a brief overview of myself as a Software Engineer. Here, visitors can see my latest projects and blog posts. The minimalist design with smooth animations provides a professional and modern impression.

### Dark Mode
The website supports dark mode for comfortable reading in low-light conditions. The transition between light and dark mode is designed with smooth transitions for a better experience.

### Blog Posts
The blog page displays technical articles with syntax highlighting for code and MDX support for more interactive content. Each article includes estimated reading time and categories for easier navigation.

## Tech Stack

This website is built using modern technology stack:

### Frontend
- **Next.js**: React framework for server-side rendering and static site generation
- **TypeScript**: For type safety and better developer experience
- **Tailwind CSS**: For flexible and maintainable styling
- **Framer Motion**: For smooth and interactive animations

### Content Management
- **MDX**: For writing content with React components
- **Contentlayer**: For transforming content into type-safe data
- **rehype/remark**: For markdown processing

### Development Tools
- **ESLint**: For code linting
- **Prettier**: For code formatting
- **Husky**: For git hooks
- **Commitlint**: For conventional commits

## Learning & Challenges

During the development of this website, I faced several challenges and learnings:

1. **Performance Optimization**: Implementation of dynamic imports and image optimization to improve performance score
2. **SEO**: Implementation of meta tags and structured data for search engine optimization
3. **Accessibility**: Ensuring the website is accessible to all users
4. **Content Management**: Using Contentlayer to manage MDX content with type safety
